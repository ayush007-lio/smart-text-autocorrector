# ✨ Smart Text Autocorrector (AI Powered Web App)

A modern **Flask-based Smart Text Autocorrect Web Application** that automatically detects and corrects spelling mistakes in sentences using a dictionary-driven NLP approach.

Built with ❤️ using Python + Flask + NLP concepts.

---

## 🚀 Live Features

✅ Corrects spelling mistakes instantly  
✅ Sentence-level autocorrection  
✅ Interactive web interface  
✅ Lightweight NLP logic  
✅ Dictionary-based correction engine  
✅ Clean responsive UI  
✅ Beginner-friendly AI project  

---

## 🧠 How It Works

This project uses a **dictionary-based Natural Language Processing technique** to:

1. Accept user text input
2. Compare words with dataset (`big.txt`)
3. Detect spelling errors
4. Suggest corrected output
5. Display results instantly on webpage

## ⚡ Installation Guide

Move into project folder

Install dependencies


pip install flask


Run the application

Open in browser


http://127.0.0.1:5000
