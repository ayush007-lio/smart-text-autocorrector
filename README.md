This project is a simple Python-based Text Autocorrect System that automatically detects and corrects spelling mistakes in user-entered sentences.

It works using a dictionary-based correction approach and processes input interactively through the command line interface.

Features:
- Corrects spelling mistakes in sentences
- Uses a custom dictionary word list
- Lightweight and beginner-friendly NLP project
- Runs in terminal/console
- Easy to extend with advanced algorithms

Technologies Used:
- Python
- Basic Natural Language Processing concepts
